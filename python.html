The Boot Camp

Learning to code is just like playing a new sport or practicing an instrument – Just get Started!! From basic programming this course will teach essential Python skills to students from 6th grades and up with fun and engaging project by developing a simple Race Computer Game using Turtle module. Turtle graphics is a popular way for introducing programming to kids.

About Python

Python is fun to learn but is also a Real programming language, Professionally Python is great for back-end web development, data analysis, artificial intelligence and scientific computing. Many developers have used Python to build productivity tools, games, and desktop apps as also for internet of things (IOT) and robotics. Today Python empowers websites and apps like Yelp.com, Twitter, YouTube and Instagram.
